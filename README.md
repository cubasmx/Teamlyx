# 📊 Teamlyx

**Solución profesional de control de tiempos para PyMEs**

Un sistema moderno y eficiente para gestionar hojas de tiempo, seguimiento de proyectos y control de recursos humanos diseñado específicamente para pequeñas y medianas empresas.

---

## ✨ Características Principales

- **⏱️ Registro de Horas**: Los empleados pueden registrar sus horas de trabajo de forma sencilla e intuitiva
- **📈 Reportes Detallados**: Genera reportes semanales y mensuales con análisis de productividad
- **👥 Gestión de Proyectos**: Asigna tareas y realiza seguimiento de tiempo por proyecto
- **💼 Integración Financiera**: Compatible con sistemas de nómina y contabilidad
- **🔒 Seguridad de Datos**: Protección avanzada y cumplimiento de privacidad
- **📱 Interfaz Amigable**: Diseño responsivo y fácil de usar
- **📊 Panel de Control**: Visualización en tiempo real de métricas clave

## 🚀 Inicio Rápido

### Requisitos Previos
- Python 3.8+
- pip (gestor de paquetes Python)

### Instalación

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/cubasmx/Teamlyx.git
   cd Teamlyx
   ```

2. **Crea un entorno virtual**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # En Windows: venv\Scripts\activate
   ```

3. **Instala las dependencias**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Ejecuta la aplicación**:
   ```bash
   python manage.py runserver
   ```

La aplicación estará disponible en `http://localhost:8000`
